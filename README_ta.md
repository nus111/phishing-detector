## மேலோட்டம்

மலேசியாவில் ஆங்கிலம் பேசாத பாதிக்கப்படக்கூடிய மக்களை (அகதிகள், வெளிநாட்டு தொழிலாளர்கள், சர்வதேச மாணவர்கள்) பாதுகாப்பதில் சிறப்பு கவனம் செலுத்தும், நிகழ்நேர ஃபிஷிங் மற்றும் மோசடி செய்திகளைக் கண்டறியும் பல மொழி AI இயக்கப்படும் அமைப்பு.

## தொழில்நுட்ப அடுக்கு

| அடுக்கு | தொழில்நுட்பம் |
|---------|--------------|
| முன்பக்கம் | React + Next.js + Tailwind CSS |
| பின்பக்கம் | Python FastAPI |
| தரவுத்தளம் | PostgreSQL |
| AI/ML | XLM-RoBERTa (Hugging Face Transformers) |
| வரிசைப்படுத்தல் | Docker + Docker Compose |

## அம்சங்கள்

- 🔍 **நிகழ்நேர செய்தி பகுப்பாய்வு** — ஃபிஷிங் குறிகாட்டிகளை சரிபார்க்க எந்த செய்தியையும் ஒட்டவும்
- 🌍 **பல மொழி ஆதரவு** — English、中文、Bahasa Melayu、தமிழ்
- ⚠️ **ஆபத்து மதிப்பீடு** — வண்ண குறியிடப்பட்ட ஆபத்து நிலைகள் (குறைந்த / நடுத்தர / அதிக) விரிவான விளக்கங்களுடன்
- 📚 **பாதுகாப்பு கல்வி** — ஃபிஷிங் விழிப்புணர்வை மேம்படுத்த உதவிக்குறிப்புகள், கட்டுரைகள் மற்றும் வினாடி வினாக்கள்
- 📊 **நிர்வாக டாஷ்போர்டு** — அச்சுறுத்தல் போக்குகள், மொழி விநியோகம், பயனாளர் பகுப்பாய்வு
- 🤖 **AI இயக்கப்படுகிறது** — துல்லியமான கண்டறிதலுக்கு XLM-RoBERTa ஆழமான கற்றல் மாதிரி

## விரைவு தொடக்கம்

```bash
# களஞ்சியத்தை குளோன் செய்யுங்கள்
git clone https://github.com/nus111/phishing-detector.git
cd phishing-detector

# சூழல் கட்டமைப்பை நகலெடுக்கவும்
cp .env.example .env
# உங்கள் அமைப்புகளுடன் .env-ஐ திருத்தவும்

# அனைத்து சேவைகளையும் தொடங்கவும்
docker-compose up -d

# அணுகல்:
# முன்பக்கம்:     http://localhost:3000
# பின்பக்கம்:     http://localhost:8000
# ML சேவை:       http://localhost:8001
# API ஆவணம்:     http://localhost:8000/docs
```

## திட்ட கட்டமைப்பு

```
phishing-detector/
├── frontend/          # Next.js முன்பக்கம் (React + Tailwind + i18n)
├── backend/           # FastAPI பின்பக்கம் (REST API + JWT அங்கீகாரம்)
├── ml/                # ML மாதிரி பயிற்சி & அனுமான சேவை
├── docs/              # API, மாதிரி, வரிசைப்படுத்தல் ஆவணம்
├── docker-compose.yml # சேவை ஒருங்கிணைப்பு
└── .env.example       # சூழல் மாறிகள் வார்ப்புரு
```

## API முடிவுப் புள்ளிகள்

| முறை | முடிவுப் புள்ளி | விளக்கம் | Auth |
|------|----------------|----------|------|
| POST | `/api/v1/auth/register` | பதிவு | ❌ |
| POST | `/api/v1/auth/login` | உள்நுழை | ❌ |
| GET | `/api/v1/auth/me` | தற்போதைய பயனாளர் | 🔒 |
| POST | `/api/v1/analyze` | செய்தியை பகுப்பாய்வு செய்யுங்கள் | 🔒 |
| GET | `/api/v1/analyze/history` | பகுப்பாய்வு வரலாறு | 🔒 |
| GET | `/api/v1/education` | கல்வி உள்ளடக்கம் | ❌ |
| POST | `/api/v1/education/quiz/submit` | வினாடி வினா சமர்ப்பிக்கவும் | 🔒 |
| GET | `/api/v1/admin/dashboard` | நிர்வாக புள்ளிவிவரங்கள் | 👑 |

> முழு API ஆவணம் `http://localhost:8000/docs` இல் கிடைக்கும் (Swagger UI)
